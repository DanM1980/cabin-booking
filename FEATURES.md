## 📊 מבנה נתונים

### Tables

```
calendar
  └─ date (PK)
  └─ status (closed/open/booked)

bookings
  └─ id (PK)
  └─ date (FK → calendar.date)
  └─ guest_name
  └─ guest_phone
  └─ guest_email
```

- פתוח לכולם (public)
